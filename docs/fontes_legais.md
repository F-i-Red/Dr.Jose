# Fontes de Legislação Portuguesa

## Como obter os textos das leis

Os textos devem ser colocados em `data/leis/` como ficheiros `.txt` ou `.pdf`.

---

## Fonte Recomendada — PGDL (gratuita e completa)

O site da **Procuradoria-Geral Distrital de Lisboa** tem todos os códigos
em texto integral e gratuito:

🔗 http://www.pgdlisboa.pt/leis/lei_main.php

### Diplomas prioritários a descarregar:

| Diploma | Link direto |
|---------|-------------|
| Constituição da República Portuguesa | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=4&tabela=leis |
| Código Penal | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=109&tabela=leis |
| Código de Processo Penal | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=199&tabela=leis |
| Código Civil | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=775&tabela=leis |
| Código do Trabalho | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=1047&tabela=leis |
| Código de Processo Civil | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=1959&tabela=leis |
| Lei de Arrendamento Urbano (NRAU) | http://www.pgdlisboa.pt/leis/lei_mostra_articulado.php?nid=1060&tabela=leis |

### Como guardar:
1. Abrir o link
2. Selecionar todo o texto (Ctrl+A)
3. Copiar (Ctrl+C)
4. Colar num ficheiro `.txt` com o nome da lei (ex: `codigo_penal.txt`)
5. Guardar em `data/leis/`

reproduzidos livremente para fins informativos e académicos.
